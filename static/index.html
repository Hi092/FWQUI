使用教程
主页圆圈
CPU：显示CPU使用率百分比，外圈是温度。点击打开CPU详情弹窗，里面有实时折线图显示CPU波动，底部有「强制降温」按钮（点击后需要输入"确认"才会重启服务器）。

内存：显示RAM使用率。点击打开内存详情弹窗，四宫格卡片显示总内存、可用、已用、缓存，下方有虚拟内存（ZRAM+磁盘Swap）信息。

磁盘：彩色弧长代表磁盘使用率，不同颜色代表不同分区。点击打开磁盘详情，可以长按拖拽调整显示顺序。

网络：显示实时网速（Mbps），不到1M显示K。点击打开网络弹窗，里面有测速按钮和历史测速记录。点击测速时网络圈会紫色脉冲闪烁。
服务管理
• 服务列表显示所有监控的服务状态，绿色圆点=运行中（带呼吸灯），红色=已停止
• 每个服务右侧有「启动/停止」按钮，点击即可操作
• 长按服务卡片（或桌面端右键）弹出快捷菜单：
  — 启动 / 停止 / 重启
  — 访问（跳转到该服务网页）
  — 改名：自定义服务显示名称，改完立即生效
  — 恢复默认名：一键恢复原始名称
• 远程类型的服务显示「控制」按钮，点击打开远程控制面板
• 点击服务名称可以直接访问该服务的网页（如FileBrowser、销售日报）
• 服务前面的绿色圆点有呼吸灯效果，全部服务正常运行时绿灯呼吸闪烁
顶部工具栏
• 第一个图标：打开终端（支持AI对话和命令执行两种模式）
• 第二个图标：查看系统日志
• 第三个图标：打开设置面板
• 第四个图标：退出登录
远程控制
服务列表中标记为「远程」类型的服务（如公司电脑），点击「控制」按钮打开远程控制面板：

状态栏：
• 显示目标设备的在线状态和IP地址

快捷操作：
• 音量：静音、30%、50%、100% 一键切换
• 锁屏、关机、重启、睡眠 — 远程控制系统
• 截屏 — 获取目标设备屏幕截图

AI远程终端：
• 支持两种模式：AI对话和命令执行
• 命令直接在目标设备上执行
• 执行结果实时显示在聊天窗口
设置 → 显示
控制主页显示哪些圆圈模块：
• CPU、内存、磁盘、网络、服务、温度、运行时间 — 各自独立开关
• 刷新频率：1-5秒可选，控制页面数据自动刷新间隔，默认3秒
• 关闭某个开关后，对应圆圈会立即从主页消失
设置 → 服务
调整服务列表的显示顺序：
• 点击↑↓按钮调整上下位置
• 点击「隐藏」可以隐藏不想看到的服务
• 设置会同步保存到服务器
设置 → 日志
控制日志页面显示哪些内容，分三组：

面板访问日志：
• 登录/登出 — 登录成功和密码错误记录
• 服务操作 — 启动/停止服务记录
• 定时任务操作 — 添加/编辑/删除定时任务
• 设置修改 — 修改密码、切换开关、服务排序等
• AI对话 — 使用AI助手对话记录（默认关闭）
• 终端命令 — 执行shell命令记录（默认关闭）
• 测速记录 — 网速测试记录
• 系统操作 — 重启、清理缓存、清理日志、修改主机名
• API调用 — 底层API调用记录（默认关闭）

外部服务日志：
• 销售日报访问 — 有人访问销售日报的记录
• FileBrowser操作 — 文件上传/下载/登录记录
• SMB文件访问 — Samba共享访问记录（默认关闭）

显示设置：
• 显示时间戳 — 日志前面的时间（默认开启）
• 显示IP地址 — 显示访问者IP（默认开启）
• 自动刷新 — 日志页面自动刷新（默认关闭）
设置 → 密码
修改面板登录密码：
• 输入当前密码 → 新密码 → 确认新密码 → 点击「确认修改」
• 密码至少4位
• 修改后需要重新登录
• 连续5次密码错误会锁定5分钟（防暴力破解）
设置 → AI服务
管理AI对话服务商：

已有服务商：
• 每个服务商显示名称和当前使用的模型
• 点击卡片选中为当前使用的服务商
• 点击「编辑」修改API地址、Key、模型
• 点击「删除」移除不用的服务商

添加新服务商：
• 点击「+ 添加新服务商」按钮
• 快速选择：提供16个预设服务商（智谱AI、OpenAI、DeepSeek、通义千问、Moonshot、百川、阶跃星辰、零一万物、讯飞星火、MiniMax、百度千帆、Groq、Together、Mistral等）
• 点击预设自动填充名称和API地址
• 输入API Key后点击「获取模型列表」自动获取可用模型
• 选择模型后点击「保存」
• API地址支持智能补全：输入域名自动补全为完整地址
设置 → 系统
系统信息：
• 主机名 — 可直接编辑，输入新名称点「保存」
• 系统 — 显示操作系统版本
• 内核 — Linux内核版本
• 架构 — CPU架构（ARM 32位/64位等）
• IP地址 — 服务器局域网IP

快捷操作：
• 重启 — 确认后重启服务器，30秒后自动刷新页面
• 清理缓存 — 清理临时文件、pip缓存、apt缓存，显示释放了多少空间
• 清理日志 — 清理journald系统日志和面板访问日志，显示释放空间
• 清空日志 — 仅清空面板操作日志（不清系统日志）
终端（AI助手）
支持两种模式，顶部切换：

AI模式：
• 跟AI对话，AI可以帮你操作服务器
• AI回复的代码块可以直接点击「运行」按钮执行
• 对话记录自动保存，下次打开还在
• 点击「清空对话」重置

命令模式：
• 直接输入shell命令执行
• 命令在后台运行，支持长时间任务
• 点击「查看结果」查看输出
系统日志
点击顶部第二个图标打开日志页面：
• 显示面板操作日志、销售日报访问、文件操作等
• 错误日志红色、警告黄色、成功绿色
• 日志内容可在「设置→日志」中精细控制
• 时间戳和IP地址可单独开关
网站监控
• 监控指定URL的可用性和响应时间
• 通过API接口添加/编辑/删除监控项
• 每个监控项显示：在线/离线状态、响应时间（ms）、SSL证书到期天数
• 配置存储在 /opt/monitor/monitors.json
网络测速
• 点击网络圆圈打开弹窗
• 点击「测速」按钮测试到Cloudflare的下载/上传速度
• 测速时网络圆圈紫色脉冲闪烁
• 测速完成后结果自动记录（最多显示最近3条）
• 测速中关闭弹窗，紫色脉冲继续闪，测速完成后自动消失
定时任务
点击服务列表中的「定时任务」打开：
• 显示当前所有定时任务
• 点击「编辑」修改时间或命令
• 点击「删除」移除任务
• 点击「+ 添加定时任务」新增
• 格式：分 时 日 月 周（如 0 8 * * * = 每天8点）
